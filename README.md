### How to deploy this project

0. folk & clone this project
1. login in netlify
2. buy a domain
3. select yours folked github repo
4. select netlify nextjs plugin
5. create a github access token
6. add `NEXT_PUBLIC_ACCESS_TOKEN` env variable in netlify
